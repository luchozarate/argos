# ARGOS

## ¿Qué es ARGOS?

ARGOS es un asistente financiero inteligente que ayuda a las personas a organizar, comprender y mejorar su economía personal mediante conversaciones naturales e inteligencia artificial.

ARGOS no es una aplicación para registrar gastos.

Su objetivo es convertirse en un compañero financiero que recuerde, aprenda, analice y recomiende acciones útiles antes de que aparezcan los problemas.

---

## Principios del proyecto

- La conversación es el centro del sistema.
- Los datos pertenecen al usuario.
- La privacidad es una prioridad.
- La IA ayuda, pero nunca reemplaza el control del usuario.
- Cada funcionalidad debe simplificar la vida del usuario.
